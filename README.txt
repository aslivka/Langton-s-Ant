Program instructions

1.	After unzipping the folder, you can compile the executable by	entering command "make".
2.	Run the program in flip with command "Project1"
3.	You will enter a menu where an option must be
	selected.
4.	Option 1 will allow a game with user defined parameters.
	values.
5.	Option 2 will allow a game with pre-defined parameters.
	values. 
6.	To quit the program, enter choice 3.
